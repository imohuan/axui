# 2026-06-03

## AxButton 修复

- 将加载状态内置到 `AxButton` 组件，新增 `loading` prop，自动渲染旋转 spinner。
- 修复 `size="icon" / "icon-lg"` 时图标未居中的问题：
  - 新增 `isIconOnly` 计算属性，在 icon-only 模式下自动隐藏默认 slot 文本，避免图标+文字挤在 24px 按钮内导致偏移。
  - 为图标 span 和 loading spinner 添加 `leading-none`，消除字体行高造成的细微偏移。
  - 预览区示例改为使用 `icon` prop 而非 slot 传入图标，确保走组件内部居中逻辑。
- `ComponentsView.vue` 中按钮示例简化，不再在 slot 里手写 spinner SVG。
