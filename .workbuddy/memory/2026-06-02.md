## AxSlider 新增标签右侧布局
- **需求**：支持将标签（labelLeft/labelRight/valueLabel）放在滑块轨道右侧显示
- **实现**：新增 `labelPosition` prop，类型 `'top' | 'right'`，默认 `'top'`（保持向后兼容）
- **布局**：
  - `top`（默认）：标签栏在轨道上方，labelLeft/labelRight 两端对齐，value 在中间
  - `right`：轨道 + 标签水平 flex 排列，轨道占 flex-1，标签在右侧 shrink-0
- **修改文件**：
  - `src/components/ui/AxSlider.vue` — 新增 prop + 双布局条件渲染
  - `skills/ax-ui-kit/assets/AxSlider.vue` — 同步更新

  - `skills/ax-ui-kit/assets/layout/ComponentsView.vue` — 同步更新
- **构建通过**（vite build ok）

## AxInput 支持多行模式（textarea）
- **需求**：AxInput 支持多行配置，并在组件列表提供对应配置选项
- **AxInput.vue 修改**：
  - 新增 `multiline` prop（默认 false），为 true 时渲染 `<textarea>` 替代 `<input>`
  - 新增 `rows` prop（默认 3），控制 textarea 行数
  - 新增 `resize` prop（'none' | 'vertical' | 'horizontal' | 'both'，默认 'vertical'）
  - textarea 使用 `TEXTAREA_SIZE_CLASSES`（不含固定高度，仅 padding + font），通过 `RESIZE_CLASSES` 控制 resize CSS
  - 新增 `textareaRef` + `onTextareaInput`；`focus()` 根据 multiline 模式选择元素
- **ComponentsView.vue 修改**：
  - inputSchema 新增：多行模式(switch)、行数(slider 1-10)、Resize(segmented)
  - inputProps 新增：showMultiline、inputRows、resize
  - 描述更新："输入框 — 支持单行/多行切换..."
  - 密码/前缀仅在单行模式下生效（`:password="inputProps.showPassword && !inputProps.showMultiline"`）
  - min-height 从 220px 升至 260px 适配 textarea
- **构建通过**（vite build ok）
